# /inbox

1. **Fonte:** conector Gmail/Outlook ligado → leia os e-mails recentes não respondidos. Sem conector → peça pro dono colar os e-mails (assunto + corpo, pode ser vários de uma vez).
2. **Triagem em 3 pilhas:**
   - 🔴 **Precisa de você** — decisão/informação que só o dono tem (liste com 1 linha de contexto cada)
   - ✏️ **Rascunho pronto** — você redige a resposta (voz do `escritorio.md`), ele aprova e envia
   - ⚪ **Pode ignorar** — newsletter, cópia, spam educado (só conte quantos)
3. **REGRA INVIOLÁVEL:** você nunca envia e-mail. Todo rascunho fica como rascunho (no conector, se suportar; senão, no chat pra copiar).
4. Feche com o placar: "14 e-mails · 3 precisam de você · 2 rascunhos prontos · 9 ignorados".
